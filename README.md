// # 🎯 Своя Игра — Web версия

Интерактивная викторина "Своя Игра" на основе классического телешоу.

## 🎮 Особенности

✅ **3 раунда** с возрастающей сложностью
✅ **Система счета** — игроки получают или теряют баллы
✅ **Таймер** — 30 секунд на ответ (с визуальным предупреждением)
✅ **Модальные окна** — удобный интерфейс для игры
✅ **Сохранение состояния** — игра сохраняется в localStorage
✅ **Горячие клавиши** — быстрое управление

## 🚀 Быстрый старт

1. **Откройте `index.html` в браузере**
   ```bash
   # Или запустите локальный сервер
   python -m http.server 8000
   # или
   npx http-server
   ```

2. **Добавьте игроков**
   - Нажмите "+ Игрок" чтобы добавить игрока
   - По умолчанию добавлены 2 игрока

3. **Выбирайте вопросы**
   - Нажимайте на цифры на доске
   - Читайте вопрос
   - Нажмите "Показать ответ" или нажмите Enter

4. **Отмечайте правильные/неправильные ответы**
   - "✓ Верно" — добавить баллы игроку
   - "✗ Неверно" — вычесть баллы игроку
   - "Пропустить" — ничего не менять

5. **Переходите к следующему раунду**
   - Нажмите стрелку вправо или закончите все вопросы

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Enter` | Показать ответ |
| `Escape` | Закрыть окно |
| `→` (вправо) | Следующий раунд |
| `↑` (вверх) | Ответ верный |
| `↓` (вниз) | Ответ неверный |

## 📊 Структура вопросов

```json
{
  "id": "q1",
  "round": 1,                    // 1, 2 или 3
  "category": "История",         // Категория вопроса
  "question": "Кто открыл Америку?",
  "answer": "Христофор Колумб",
  "type": "normal"               // или "final"
}
```

## 🎯 Правила игры

### Раунды
- **Раунд 1**: 100, 200, 300, 400, 500 баллов
- **Раунд 2**: 200, 400, 600, 800, 1000 баллов
- **Раунд 3**: 300, 600, 900, 1200, 1500 баллов

### Финальный вопрос
- После Раунда 3 выводится финальный вопрос
- Ведущий показывает ответ
- Финальный раунд завершает игру

## 🎨 Кастомизация

### Изменить количество категорий
В `game.js`, измените `CONFIG.CATEGORY_COUNT`:
```javascript
CATEGORY_COUNT: 6  // Количество категорий на доске
```

### Изменить время ответа
```javascript
TIMER: 30  // Секунды на ответ
```

### Изменить стоимость вопросов
```javascript
ROUND_VALUES: {
    1: [100, 200, 300, 400, 500],
    2: [200, 400, 600, 800, 1000],
    3: [300, 600, 900, 1200, 1500]
}
```

## 💾 Сохранение данных

Игра автоматически сохраняет состояние в `localStorage` под ключом `svoya_game_v5`:
- Список игроков и их счет
- Текущий раунд
- Использованные вопросы

Чтобы начать заново, нажмите "Сброс" или очистите localStorage:
```javascript
localStorage.removeItem("svoya_game_v5");
```

## 🔧 Разработка

### Архитектура

```
GameEngine (class)
├── init() — загрузка вопросов
├── bootstrap() — инициализация
├── buildRound() — построение раунда
├── openQuestion() — показ вопроса
├── correct/wrong() — подсчет баллов
└── saveState/loadState() — сохранение
```

### Добавить новые вопросы

Добавьте объекты в `questions.json`:
```json
{
  "id": "q999",
  "round": 1,
  "category": "Мясо",
  "question": "Сколько ног у курицы?",
  "answer": "2 ноги",
  "type": "normal"
}
```

## 🐛 Известные проблемы

- На мобильных устройствах кнопки могут быть тесными
- Таймер может отставать на медленных устройствах
- localStorage доступен только в HTTPS или localhost
